This readme contains information for setting up a development environment on visual studio

1) For setting up cuda compiler in visual studio visit the following link for a step by step guide:
	http://blog.cuvilib.com/2011/02/24/how-to-run-cuda-in-visual-studio-2010/

2) The second dependency in opencv. Download opencv 2.2 for vs2010 from the following link
	http://sourceforge.net/projects/opencvlibrary/files/opencv-win/2.2/

3) The third step is setting up of opencv in visual studio. Goto the following link:
	http://opencv.willowgarage.com/wiki/VisualC%2B%2B
   Under the section "Building your own projects using OpenCV 2.2 in Visual Studio" are the 
   detailed steps for setting up opencv2.2 in visual studio.

4) All the nvidia graphic cards with compute capability >=1.3 will be bale to compile and run 
   code. So kindly check the CC for you GC.
